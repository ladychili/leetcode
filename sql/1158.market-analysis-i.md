# 1158.(Medium) 市场分析 I

Table: Users

```text
+----------------+---------+
| Column Name    | Type    |
+----------------+---------+
| user_id        | int     |
| join_date      | date    |
| favorite_brand | varchar |
+----------------+---------+
此表主键是 user_id，表中描述了购物网站的用户信息，用户可以在此网站上进行商品买卖。
```

Table: Orders

```text

| Column Name   | Type    |
+---------------+---------+

| order_date    | date    |
| item_id       | int     |
| buyer_id      | int     |
| seller_id     | int     |
+---------------+---------+
此表主键是 order_id，外键是 item_id 和（buyer_id，seller_id）。
```

Table: Item

```text
+---------------+---------+
| Column Name   | Type    |
+---------------+---------+
| item_id       | int     |
| item_brand    | varchar |
+---------------+---------+
此表主键是 item_id。
```

请写出一条SQL语句以查询**每个用户的注册日期** 和 **在 2019 年作为买家的订单总数**。

查询结果格式如下：


## Solution 

```sql
select 
    user_id buyer_id, 
    max(join_date) join_date,
    count(distinct order_id) orders_in_2019
from Users u LEFT JOIN Orders o 
    on u.user_id = o.buyer_id and LEFT(order_date,4)='2019'
group by 1
```

# 176.\(Easy\) 第二高的薪水

#### Query1  

{% code title="" %}
```sql
SELECT DISTINCT Salary AS SecondHighestSalary
FROM Employee
ORDER BY Salary DESC
LIMIT 1,1
```
{% endcode %}

#### 无法返回`NULL`

#### 

#### Query2 

{% code title="" %}
```sql
SELECT (
    SELECT DISTINCT Salary 
    FROM Employee
    ORDER BY Salary DESC
    LIMIT 1,1
) AS SecondHighestSalary
```
{% endcode %}

#### 在 query1外嵌套另一个SELECT子句，即可在空表适返回`NULL`

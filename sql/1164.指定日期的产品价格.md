# 1164.(Medium) 指定日期的产品价格


产品数据表: Products
```
+---------------+---------+
| Column Name   | Type    |
+---------------+---------+
| product_id    | int     |
| new_price     | int     |
| change_date   | date    |
+---------------+---------+
这张表的主键是 (product_id, change_date)。
这张表的每一行分别记录了 某产品 在某个日期 更改后 的新价格。
```

写一段 SQL来查找在 2019-08-16 时全部产品的价格，假设所有产品在修改前的价格都是 10。

查询结果格式如下例所示：
```
Products table:
+------------+-----------+-------------+
| product_id | new_price | change_date |
+------------+-----------+-------------+
| 1          | 20        | 2019-08-14  |
| 2          | 50        | 2019-08-14  |
| 1          | 30        | 2019-08-15  |
| 1          | 35        | 2019-08-16  |
| 2          | 65        | 2019-08-17  |
| 3          | 20        | 2019-08-18  |
+------------+-----------+-------------+

Result table:
+------------+-------+
| product_id | price |
+------------+-------+
| 2          | 50    |
| 1          | 35    |
| 3          | 10    |
+------------+-------+
```


## Solution

1. 先找出每个产品2019-08-16之前的价格 p2
2. left join, group by取p2的价格，ifnull补10

``` sql 
select p1.product_id, ifnull(p2.new_price,10) price
from Products p1 left join 
    (select product_id, new_price 
    from Products
    where (product_id,change_date) in 
        (select product_id, max(change_date) nearest_date
        from Products 
        where change_date<='2019-08-16'
        group by product_id) ) p2 
    on p1.product_id = p2.product_id
group by p1.product_id

```
